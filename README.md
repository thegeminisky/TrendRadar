## 🚀 快速开始
**[查看最新官方文档](https://github.com/sansan0/TrendRadar?tab=readme-ov-file)**
```
本仓库将钉钉机器人推送的鉴权方式切换为加签
如选择钉钉机器人作为推送渠道，与官方教程有以下差异
1.在创建钉钉机器人时需要勾选“加签”
2.在配置dingtalk_url时需要在相应位置配置dingtalk_secret
3.在Github Action中运行需要额外添加"DINGTALK_SECRET"
```
